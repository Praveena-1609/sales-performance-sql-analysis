# Sales Performance & Customer Order Analysis (SQL)

## Overview
This project analyzes retail sales data using SQL to generate insights on customer behavior, product performance, and revenue trends.

## Key Analysis
- Top customers and products
- Sales by region
- Monthly sales trends
- Month-over-month growth
- Shipment delay analysis
- Customer segmentation

## Tools Used
- MySQL
- SQL (Joins, CTEs, Window Functions)

## Dataset
- 4 relational tables: customers, products, orders, order_items
- ~300+ records

## Key Skills Demonstrated
- Query Optimization
- Data Aggregation
- Window Functions
- Business KPI Analysis
